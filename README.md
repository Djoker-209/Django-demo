# Django Project

## Setup
```bash
git clone https://github.com/Djoker-209/Django-demo.git
cd Django-demo
pip install -r requirements.txt
python manage.py migrate
python manage.py runserver
```

Create `.env` file:
```
SECRET_KEY = 'django-insecure-change-me-in-production'
DEBUG = True
